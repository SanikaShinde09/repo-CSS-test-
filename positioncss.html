Using position: static;

An element with position: static; is always positioned according to the normal flow of the page:

this div is static position: static;

Using position: relative;

An element with position: relative; is always positioned according to the normal flow of the page:

This div is relative position: relative;

Using position: fixed;

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled:

This div is fixed position: fixed;
